Q-2 What is the use of isNaN function?

Ans :

The isNaN function in JavaScript is used to check if a value is "Not a Number."

It returns true if the provided value is not a valid number

Example :

<script>
console.log(isNaN(123)); // false
console.log(isNaN("hello")); // true
console.log(isNaN("hello")); // true
</script>
