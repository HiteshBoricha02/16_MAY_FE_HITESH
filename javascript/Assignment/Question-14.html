Q-14 What are all the types of Pop up boxes available in JavaScript?

Ans :

Three types of pop up boxes in JavaScript

1. Alert Box

2. Confirm Box

3. Prompt Box

Example:

            1. Alert Box 
            alert("This is an alert box!");

        -------------------------------------------------------

            2. confirm Box 
            confirm("This is javascript assignment");

        -------------------------------------------------------


            3. Prompt Box
            Prompt("This is javascript");

        -------------------------------------------------------
