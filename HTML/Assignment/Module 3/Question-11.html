elements are two type     :   1) block Level        2) Inline


1) Block Level  :

Block-level elements start on a new line and take up the full available width by default. They create a "block" of content and typically represent structural elements in the document, such as paragraphs, headings, dividers, lists, and sections. Examples of block-level elements include <p>, <div>, <h1> to <h6>, <ul>, <ol>, and <section>. Block-level elements can contain other block-level and inline elements. They can have their width, height, margins, padding, and other CSS properties adjust

2) Inline elements  :

Inline elements do not start on a new line; they flow alongside other elements within the same line. They only occupy the space required by their content, without creating line breaks. Examples of inline elements include <span>, <a>, <strong>, <em>, <img>, <input>, <label>, and others. Inline elements do not have width, height, margins (top/bottom), or paddings (top/bottom) applied to them by default. However, you can adjust horizontal margins and paddings.
