Three types of lists
1) ordered list
2) un ordered list
3) Data list


1) Ordered list (<ol>): Ordered lists are used when the order of the items is important. Each item is represented by an <li> (list item) element, and the items are numbered sequentially by default. You can use the type attribute to specify a different numbering style (such as lowercase letters, uppercase letters, or Roman numerals).

Example :

    Week:
  1. Sunday
  2. Monday
  3. Tuesday
  4. Wednesday
  5. Thursday
  6. Friday
  7. Saturday


2) Unordered lists (<ul>): Unordered lists are used when the order of the items doesn't matter. Each item is represented by an <li> element, and the items are typically displayed with bullet points by default. You can use CSS to customize the bullet style.

    Example :

    color:
  • Red
  • Blue
  • green
  • white
  • yellow


3) Definition lists (<dl>): Definition lists are used to present a list of terms along with their definitions. Each term is represented by a <dt> (definition term) element, and each definition is represented by a <dd> (definition description) element.
