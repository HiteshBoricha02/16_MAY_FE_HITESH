how is cell padding different cell spacing ? With Example

Cell padding and cell spacing are two different concepts in HTML that are used to control the spacing around the content within table cells.

Cell Padding :

Cell padding refers to the space between the content of a cell and its border. It allows you to specify the amount of space to be added inside each cell. The padding is applied to all four sides (top, bottom, left, and right) of the cell's content. It is set using called "padding."

Cell Spacing :

Cell spacing refers to the space between adjacent cells within a table. It allows you to specify the amount of space between cells. The spacing is applied around each cell in the table. It is set using the HTML attribute called "cellspacing" within the "table" tag.
