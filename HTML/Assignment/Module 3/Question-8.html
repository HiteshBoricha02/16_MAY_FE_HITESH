What are the Various Formatting tags in HTML

HTML provides various formatting tags that allow you to apply formatting and styling to different parts of your content. Here are some commonly used formatting tags in HTML

1) <b> and <strong>:
                                  <b> is used to make the enclosed text bold.
                                 <strong> is used to indicate strong importance or emphasis to the enclosed text. It is usually rendered as bold by default.

2) <em>:
               <em> is used to indicate emphasis on the enclosed text. It is typically rendered as italic by default.

3) <u> :
             <u> is used to underline the enclosed text

4) <sub> and <sup>:
                         <sub> is used to display the enclosed text as subscript
                         <sup> is used to display the enclosed text as superscript

5) <code>:
                <code> is used to represent a block of code or inline code snippets. It is typically rendered in a monospace font

6) <blockquote>:
                          <blockquote> is used to indicate a block quotation or a section that is quoted from another source. It often renders with an indentation or blockquote styling.
