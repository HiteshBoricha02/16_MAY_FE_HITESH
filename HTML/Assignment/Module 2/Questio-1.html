Website :    A website is a collection of interconnected webpages that are typically hosted on a single domain or subdomain. It represents a location on the World Wide Web (WWW) that
                    can be accessed by users through a web browser. Websites can contain various types of content, such as text, images, videos, and interactive elements.


Webpage :    A webpage is a single document or file within a website. It is displayed in a web browser and contains specific content such as text, images, multimedia, and hyperlinks.
                       Webpages are written using programming languages like HTML (Hypertext Markup Language) and styled with CSS (Cascading Style Sheets).


Web browser :   A web browser is a software application that enables users to access and view websites on the internet. Popular web browsers include Google Chrome, Mozilla Firefox,
                           Microsoft Edge, Safari, and Opera. Web browsers interpret and display HTML, CSS, JavaScript, and other web technologies to present webpages to users.


Web server :    A web server is a computer program or hardware device that hosts websites and delivers web content to web browsers upon request. It stores website files, such as HTML,
                          CSS, images, and other assets, and responds to HTTP (Hypertext Transfer Protocol) requests from web browsers by serving the requested files.


HTML  :    HTML Stands For(Hypertext Markup Language) is the standard markup language used for creating the structure and content of webpages. It consists of a series of elements or
                  tags that define the structure and semantics of the content on a webpage. HTML tags are used to mark up headings, paragraphs, images, links, forms, and other elements.


CSS  :    CSS Stands For(Cascading Style Sheets) is a style sheet language used for describing the presentation and visual appearance of webpages written in
               HTML or XML (eXtensible Markup Language). It defines how HTML elements should be displayed on the screen, including layout, colors, fonts, and other visual properties. CSS
               allows web designers to separate the structure (HTML) from the presentation (CSS) of a webpage, making it easier to maintain and update the design.
