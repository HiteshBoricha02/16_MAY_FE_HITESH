Three type of css (Cascading Style Sheets)

1) inline css

2) internal css

3) External css

1) inline css :

this method, CSS styles are directly applied to HTML elements using the "style" attribute.


2) internal css :

internal CSS, the CSS code is placed within the HTML file, typically in the <head> section. It is enclosed within <style>


3) External css

CSS code is placed in a separate external file with a .css extension. The CSS file is then linked to the HTML file using the <link> tag in the <head> section.
