HTML :
HTML stands for (Hyper text markup language) it is markup language use for creating basic structure of web page

CSS :
CSS stands for (Cascading Style Sheets) It is a style sheet language used for describing the presentation and visual appearance of a document written in HTML CSS allows you to define the layout, colors, fonts, and other visual properties of HTML elements.

JavaScript :
JavaScript is a high-level programming language primarily used for creating dynamic and interactive web content. It is often referred to as the "language of the web" because it is widely supported by web browsers and allows developers to add functionality to websites.

J Query
jQuery is a fast, lightweight, and feature-rich JavaScript library. It simplifies HTML document traversal, event handling, animation, and interaction with the Document Object Model (DOM). jQuery makes it easier to write JavaScript code by providing a concise and intuitive syntax.

Bootstrap :
Bootstrap is a popular open-source framework for building responsive and mobile-first websites and web applications. It provides a collection of pre-designed HTML, CSS, and JavaScript components and utilities that simplify and accelerate the process of front-end development.
