Q-3 What is the difference between CSS2 and CSS3?

Selectors:

CSS2 has basic selectors like element, class, and ID selectors.
CSS3 introduces more advanced selectors like attribute, child, and pseudo-class selectors, of elements.

Animations and Transitions:

CSS2 has basic support for animations and transitions.
CSS3 introduces keyframe animations and transitions with more advanced timing and easing functions.

Shadows:

CSS2 has limited support for creating shadows.
CSS3 introduces the box-shadow property for adding shadows to elements, providing more control over shadow effects.

Text Effects:

CSS2 has basic text styling options.
CSS3 introduces advanced text effects like text shadows, multiple fonts, and text wrapping.
