Q-4 Name a few CSS style components

Color:

CSS allows you to set the color of text, backgrounds, borders, and other elements on a webpage.

Font:

You can control the font family, size, style (like bold or italic), and weight (like normal or bold) of text using CSS.

Spacing:

CSS lets you adjust the spacing around elements, such as margins (space outside an element) and padding (space inside an element).

Text Alignment:

You can align text within elements, like left, center, or right alignment.

Animations:

CSS allows you to create animations, like moving elements or changing their size or color over time.

Transitions:

You can add smooth transitions to elements, like fading in or out when you hover over them, using CSS.

Borders:

CSS allows you to create borders around elements with options for border width, style (like solid or dashed), and color.
