Q-13 What are the ways to integrate CSS as a web page?

Css has tree type to integrate in web page inline css internal css and last external css

inline CSS:

You can include CSS directly within the HTML document using the "style" attribute. This applies styles only to the specific HTML element

internal CSS:

You can CSS styles within the <style> element in the HTML document's <head > section. These styles apply to elements throughout the page.

External CSS:

link an external CSS file to your HTML document. Create a separate .css file, and then use the <link> element in the HTML document's <head> section to connect the CSS file.
