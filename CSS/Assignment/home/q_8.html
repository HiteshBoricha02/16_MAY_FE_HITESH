Question-8 : What is the use of the background-position property?

Ans:

Background position defines the background image's location by top, right, bottom, left, center, etc.

(BACKGROUND-POSITION: CENTER;)

Exampal :


.image div {
    height: 250px;
    width: 50%;
    background: url("../image/bg_img.jpeg");
    background-size: cover;
    background-repeat: no-repeat;
    background-position: left;
}
