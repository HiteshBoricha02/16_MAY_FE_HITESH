Q-15 What are the external style sheets?

separate CSS files that contain all the styling instructions for a website. Instead of placing CSSdirectly within an HTML document,
you create an external .css file and link it to your HTML pages. This way, you can apply the same styles to multiple web pages, promoting consistency and making it easier to manage your website's design.

Example:

<!DOCTYPE html>

<html>

<head>

<link rel="stylesheet" type="text/css" href="styles.css">

</head>

<body>

<h1>This heading is styled using the external CSS file.</h1>

<p>This paragraph also uses styles from the external CSS.</p>

</body>

</html>
