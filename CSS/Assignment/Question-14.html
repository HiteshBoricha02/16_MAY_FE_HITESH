Q-14 What is embedded style sheets?

Embedded style sheets, in simple language, are CSS styles that you include directly within an HTML document. These styles are placed inside the
<style> element, which typically goes in the <head> section of your HTML document.

Example:

<style>

div{

color: rgb(17, 71, 96);

font-size: 20px;

font-family: sans-serif;

margin-top: 2%;

border-bottom: 2px dotted black;

}

p{

color: rgb(17, 71, 96);

font-size: 20px;

font-family: sans-serif;

margin-top: 2%;

border-bottom: 2px dotted black;

}

</style>
