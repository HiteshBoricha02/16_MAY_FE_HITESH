Q-6 How can the background color of an element be changed?

Select the element you want to change the background color of, like a paragraph or a div.
In your CSS code, write the element's name (like p for paragraphs) followed by curly braces {}.
Inside the curly braces, add the property background-color: followed by the color you want. You can use color names (e.g., "red" or "blue") or color codes (e.g., "#FF0000" for red).
