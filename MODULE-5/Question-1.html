What are the new tags added in HTML5?

Ans :

HTML5 introduced several new tags compared to previous versions of HTML. Some of the notable new tags added in HTML5 include:

Tag Description
<header> Represents the introductory content of a section or a page.
<nav> Represents the introductory content of a section or a page.
<section> Represents a standalone section of content.
<article> Defines an independent, self-contained piece of content.
<aside> Represents content that is tangentially related to the main content, often presented as sidebars.
<footer> Represents the footer of a section or a page.
<figure> Represents self-contained content, such as images, diagrams, or code snippets, that is referenced in the main content.
<figcaption> Defines a caption for a <figure> element.
<main> Specifies the main content of a document or a section.
<time> Represents a specific time or a range of time.
<progress> Displays the progress of a task, such as file upload or download.
<meter> Represents a scalar measurement within a known range.
<video> Embeds video content into an HTML document.
<audio> Embeds audio content into an HTML document.
<canvas> Provides a drawing space for JavaScript and is used for rendering graphics, animations, or interactive media.
