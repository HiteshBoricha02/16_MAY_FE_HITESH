How to embed audio and video in a webpage?

Ans :

The HTML5 <audio> and <video> tags make it simple to add media to a website. You need to set src attribute to identify the media source and include a controls attribute so the user can play and pause the media.

Example

Html

<section class="video">
<video src="../image/video.mp4" loop controls poster="../image/poster.webp">
<source src="../image/video.mp4" type="video/mp4">
</video>
</section>

Style

section .box{
width: 50%;
margin: auto;
border: 2px solid blue;
}

section .box h2{
padding: 10px;
background: blue;
color: #fff;
}

section .box h4{
padding: 10px 0px 0px 20px;
font-size: 20px;
font-family: 'Franklin Gothic Medium', 'Arial Narrow', Arial, sans-serif;
}

Audio Tag

Example

<section class="audio">
<audio src="../image/audio.mp3" controls autoplay muted>
<source src="../image/audio.mp3" type="audio/mp3">
</audio>
</section>
